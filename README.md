<h2 name='terminal'> Comandos no terminal </h2>

Acessar o Mongo
````shel
Mongo
````

Ligar o Mongo
````shel
Mongod
````
Monstrar bancos na máquina
````shel
 show dbs
````
Monstrar as collection
````shel
show collections
````
Ver Conteúdo dentro da collection
````shel
db.<nome da collection>.find()
````
